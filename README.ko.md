# dd

> **크게 복사해도 대화는 가볍게 — 통째로 말고 클립보드만 떨궈요.**

`dd`는 지금 클립보드에 있는 걸 — 스크린샷, 레퍼런스 이미지, 긴 에러 로그 — 대화에 통째로 쏟아붓지 않고 Claude Code에 짧은 참조로 건넵니다. 원문은 로컬 파일에 남고, 대화엔 (민감정보 마스킹된) 짧은 요약만 들어갑니다. 세션이 가볍고 빠르게 유지되고, 터미널이 못 붙여넣는 이미지도 `dd`가 잡습니다.

## 빠른 시작

### 1. 마켓플레이스 추가

```
/plugin marketplace add https://github.com/fivetaku/gptaku_plugins.git
```

### 2. 설치

```
/plugin install dd
```

### 3. Claude Code 재시작

(새 명령은 재시작해야 등록됩니다.)

### 4. 사용법

클립보드에 뭔가 올린 뒤, `/dd`(또는 `/ㅇㅇ`)에 요청을 붙여 입력하면 됩니다 — **붙여넣기 안 해도 돼요**:

- **이미지 — Windows:** `Win`+`Shift`+`S`로 캡처 → `/dd 왜 이렇게 보여?` (캡처가 이미 클립보드에 있음)
- **이미지 — macOS:** `Ctrl`+`Shift`+`Cmd`+`4`로 캡처 — `Ctrl`을 더하면 파일이 아니라 **클립보드**로 들어감 — → `/dd 이 레퍼런스처럼 만들어줘`
- **긴 텍스트 / 에러 로그:** 복사 → `/dd 무슨 에러야?` — 대화창에 긴 텍스트가 통째로 안 들어가서 세션이 가볍게 유지됨
- **요청 없이:** 그냥 `/dd` — 클립보드를 읽고 대화 맥락을 이어 처리

> **macOS 팁:** 그냥 `Shift`+`Cmd`+`4`가 항상 클립보드로 가게 하려면, 스크린샷 도구막대(`Shift`+`Cmd`+`5`) → **옵션** → **저장 위치: 클립보드**로 바꾸면 됩니다. 그러면 `Ctrl` 없이도 돼요.

한글 입력 상태에선 `/dd`를 그대로 쳐도 `/ㅇㅇ`로 동작해요. 한영 전환 불필요.

## 왜 dd인가

- **긴 붙여넣기는 대화를 비대하게 만듭니다.** 긴 로그나 파일을 대화에 붙이면 계속 남아 자리를 차지하고 매 턴마다 다시 읽힙니다 — 세션이 느려지고, 비싸지고, 따라가기 어려워지죠. `dd`는 원문을 로컬 파일에 두고 짧은 요약만 주입하며, 나머지는 크기에 따라 필요할 때만 읽습니다.
- **이미지 붙여넣기가 항상 되는 건 아닙니다.** 환경에 따라 Claude Code 세션이 붙여넣은 이미지를 아예 못 받기도 합니다. `dd`는 붙여넣을 필요가 없어요 — 클립보드에 캡처하고 `/dd`만 치면 끝. (복사한 이미지 *파일*도 아이콘이 아니라 실제 그림으로 해석합니다.)
- **보일러플레이트 제거.** "방금 복사한 에러", "위 레퍼런스 이미지" 같은 설명을 더는 안 써요. 그냥 `/dd` + 요청.
- **엉뚱한 클립보드 방지.** 클립보드엔 시각이 없어서, `dd`는 무엇을 잡았는지 보여주고 요청과 동떨어져 보이면 작업 전에 확인합니다.

## 작동 방식

```
복사 / 스크린샷
   → /dd [요청]
   → dd_clipboard.py가 현재 클립보드를 ~/dd/<날짜>/<id>/ 에 캡처
       · 텍스트 → content.txt   이미지 → image.png   + manifest.json
   → Claude는 manifest 요약만 읽음 (원문 전체 X)
   → 뭘 잡았는지 보여주고, 이상하면 확인, 맞으면 바로 작업
```

OS 클립보드는 가장 최근 복사 1개만 담습니다(이력 없음). `dd`는 항상 그 현재 항목을 잡고, preview에서 민감정보를 가리며, 7일 지난 캡처는 자동 정리합니다.

## 주요 기능

| 기능 | 설명 |
|------|------|
| 텍스트·이미지 캡처 | macOS / Windows / WSL / Linux 클립보드, 텍스트 또는 이미지 |
| instruction 패턴 주입 | 매 `/dd`마다 `${CLAUDE_PLUGIN_ROOT}`로 캡처 — 누구나 동작 |
| 확인 게이트 | 클립보드가 요청과 무관하거나 애매할 때만 작업 전 확인 |
| 지연 읽기 | `size_class`별로 읽어 거대한 붙여넣기가 컨텍스트를 안 덮침 |
| 민감정보 마스킹 | `api_key`, `Bearer`, `sk-`, `ghp_`, `xoxb-` 등 preview에서 가림 |
| 자동 정리 | `DD_RETENTION_DAYS`(기본 7일) 지난 캡처를 실행 시 삭제 |

## 요구사항

- [Claude Code](https://docs.anthropic.com/claude-code) CLI
- Python 3 (표준 라이브러리만)
- **macOS**: 바로 동작 (`pbpaste` / `osascript`)
- **Windows / WSL**: PowerShell(기본 내장) — 사용자 검증
- **Linux**: 클립보드 도구 설치(`xclip` / `wl-clipboard`) + GUI 세션 필요
